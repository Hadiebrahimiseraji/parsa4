# Hemato Lesson Site (Multi-page)

این پوشه یک **درسنامه چندصفحه‌ای** از محتوای `data/hemato_full_structured.json` است که برای حالت درسنامه **نرمال‌سازی** شده است.

## ساختار
- `index.html` فهرست درس‌ها
- `pages/` هر درس یک صفحه (نام‌گذاری: 01.html, 02.html, ...)
- `assets/` استایل و اسکریپت
- `data/` فایل خام + `book.json` نسخه نرمال‌سازی‌شده

## اجرا
کافی است `index.html` را باز کنید (یا با GitHub Pages منتشر کنید).
